ahooks

ahooks 是一个 React Hooks 库，致力提供常用且高质量的 Hooks。

特点：

  1. 支持按需加载
  2. 用法丰富多样

API

Async

useRequest: 发情请求

Table

  1. useAntdTable: 封装了常用的 antd Form 与 antd Table 联动逻辑，并且同时支持 antd V3 和 V4。

  2. useFusionTable: 封装了常用的 Fusion Form 与 Fusion Table 联动逻辑。

UI

  1. useDrop & useDrag: 一对帮助你处理在拖拽中进行数据转移的 hooks

  2. useDynamicList: 一个帮助你管理列表状态，并能生成唯一 key 的 Hook。

  3. useSelections: 常见联动 checkbox 逻辑封装，支持多选，单选，全选逻辑，还提供了是否选择，是否全选，是否半选的状态。

  4. useVirtualList: 提供虚拟化列表能力的 Hook，用于解决展示海量数据渲染时首屏渲染缓慢和滚动卡顿问题。

SideEffect

  1. useDebounce: 用来处理防抖值的 Hook。

  2. useDebounceFn: 用来处理防抖函数的 Hook。

  3. useInterval: 一个可以处理 setInterval 的 Hook。

  4. useThrottle: 用来处理节流值的 Hook。

  5. useThrottleFn: 用来处理节流函数的 Hook。

  6. useTimeout: 一个可以处理 setTimeout 计时器函数的 Hook。

LifeCycle

  1. useDebounceEffect: 为 useEffect 增加防抖的能力。

  2. useMount: 只在组件 mount 时执行的 hook。

  3. useThrottleEffect: 为 useEffect 增加节流的能力。

  4. useTrackedEffect: 在 useEffect 的基础上，追踪触发 effect 的依赖变化。

  5. useUnmount: 只在组件 unmount 时执行的 hook。

  6. useUnmountedRef: 获取当前组件是否已经卸载的 hook，用于避免因组件卸载后更新状态而导致的内存泄漏

  7. useUpdate: 强制组件重新渲染的 hook。

  8. useUpdateEffect: 一个只在依赖更新时执行的 useEffect hook。

  9. useUpdateLayoutEffect: 一个只在依赖更新时执行的 useLayoutEffect Hook。

State

  1. useUrlState: 一个同步组件内部状态和 query 参数的 hook。

  2. useBoolean: 优雅的管理 boolean 值的 Hook。

  3. useControllableValue: 在某些组件开发时，我们需要组件的状态即可以自己管理，也可以被外部控制，useControllableValue 就是帮你管理这种状态的 Hook。

  4. useCookieState: 一个可以将状态持久化存储在 cookie 中的 Hook 。

  5. useCountDown: 一个用于管理倒计时的 Hook。

  6. useCounter: 一个可以管理 count 的 Hook。

  7. useHistoryTravel: 优雅的管理状态变化历史，可以快速在状态变化历史中穿梭 - 前进跟后退。

  8. useLocalStorageState: 一个可以将状态持久化存储在 localStorage 中的 Hook 。

  9. useMap: 一个可以管理 Map 类型状态的 Hook。

  10. useNetwork: 一个可以管理网络连接状态的 Hook。

  11. usePrevious: 保存上一次渲染时状态的 Hook。

  12. useSessionStorageState: 一个可以将状态持久化存储在 sessionStorage 中的 Hook。

  13. useSet: 一个可以管理 Set 类型状态的 Hook。

  14. useSetState: 管理 object 类型 state 的 Hooks，用法与 class 组件的 this.setState 基本一致。

  15. useToggle: 用于在两个状态值间切换的 Hook。

  16. useWebSocket: 用于处理 WebSocket 的 Hook。

  17. useWhyDidYouUpdate: 帮助开发者排查是什么改变导致了组件的 rerender。

Dom

  1. useClickAway: 优雅的管理目标元素外点击事件的 Hook。

  2. useDocumentVisibility: 可以获取页面可见状态的 Hook。

  3. useEventListener: 优雅使用 addEventListener 的 Hook。

  4. useEventTarget: 常见表单控件(通过 e.target.value 获取表单值) 的 onChange 跟 value 逻辑封装，支持自定义值转换和重置功能。

  5. useExternal: 一个用于动态地向页面加载或卸载外部资源的 Hook。

  6. useFavicon: 用于设置与切换页面 favicon。

  7. useFullscreen: 一个用于处理 dom 全屏的 Hook。

  8. useHover: 一个用于追踪 dom 元素是否有鼠标悬停的 Hook。

  9. useInViewport: 一个用于判断 dom 元素是否在可视范围之内的 Hook。

  10. useKeyPress: 一个优雅的管理 keyup 和 keydown 键盘事件的 Hook，支持键盘组合键，定义键盘事件的 key 和 keyCode 别名输入 。

  11. useMouse: 一个跟踪鼠标位置的 Hook

  12. useResponsive: 获取响应式信息。

  13. useScroll: 获取元素的滚动状态。

  14. useSize: 一个用于监听 dom 节点尺寸变化的 Hook。

  15. useTextSelection: 实时获取用户当前选取的文本内容及位置。

  16. useTitle: 用于设置页面标题的 Hook。

Advanced

  1. useCreation: useCreation 是 useMemo 或 useRef 的替代品。因为 useMemo 不能保证被 memo 的值一定不会被重计算，而 useCreation 可以保证这一点。

  2. useEventEmitter： 在多个组件之间进行事件通知有时会让人非常头疼，借助 EventEmitter ，可以让这一过程变得更加简单。

  3. useLockFn： 用于给一个异步函数增加竞态锁，防止并发执行。

  4. usePersistFn： 持久化 function 的 Hook。

  5. useReactive： 提供一种数据响应式的操作体验,定义数据状态不需要写 useState , 直接修改属性即可刷新视图。

  6. useSafeState： 用法与 React.useState 完全一样，但是在组件卸载后异步回调内的 setState 不再执行，避免因组件卸载后更新状态而导致的内存泄漏
